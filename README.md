# Alinhatta - Consultoria em Licitações

Site institucional da Alinhatta - Consultoria especializada em licitações públicas.

## Sobre

A Alinhatta oferece consultoria especializada com mais de 4 anos de experiência em licitações públicas, transformando empresas comuns em competidoras vitoriosas em editais públicos.

## Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)
- Vercel (Deploy)

## Estrutura

```
alinhatta-site/
├── index.html      # Página principal do site institucional
├── vercel.json     # Configuração de deploy no Vercel
└── logo.svg        # Logo da empresa
```

## Deploy

O site está configurado para deploy no Vercel. O arquivo `vercel.json` dentro da pasta `alinhatta-site` contém as configurações necessárias.

### Deploy Manual

1. Acesse [vercel.com](https://vercel.com)
2. Adicione novo projeto
3. Selecione a pasta `alinhatta-site`
4. Configure o domínio: `www.alinhatta.com.br`

## Desenvolvimento Local

Para testar localmente:

```bash
cd alinhatta-site
python -m http.server 8000
```

Acesse: http://localhost:8000
